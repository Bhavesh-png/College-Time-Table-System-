## ✨ Features

### 📅 Faculty-Centric Timetable
- Faculty-first layout (rows = faculty, columns = days)  
- Color-coded schedule chips  
- Displays subject, timing & duration  
- Weekly workload tracking (`12h / 40h`)  
- Overload warning system  
- Unique avatars for faculty  

---

### 🕰️ Dashboard System
- Animated analog clock  
- Tear-off calendar widget  
- Live statistics:
  - Active classes  
  - Faculty count  
  - Room usage  

---

### 🔒 Authentication
- Modern login/register UI  
- Secure login system  
- Password recovery (security questions)  
- Admin panel access  

---

### 🛠️ Smart Scheduling Engine
- Start & end time precision  
- Conflict detection:
  - Prevents double booking  
- Right-click delete functionality  
- Real-time UI updates

## 🚀 Getting Started

### 📌 Prerequisites
- JDK 17+
- Maven 3.8+

### ⚡ Installation & Run

```bash
# Clone the repository
git clone https://github.com/BHavesh-png/College-Time-Table-System.git

# Navigate into project
cd College-Time-Table-System

# Build project
mvn clean install

# Run application
mvn exec:java -Dexec.mainClass="com.college.App"
